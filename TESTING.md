### 3. Alias Installation Tests

#### 3.1 Alias File Creation
```bash
# Run ccodex (creates aliases automatically)
npx -y ccodex --status

# Check alias file was created
cat ~/.oh-my-zsh/custom/ccodex-alias.zsh
```

**Expected Content**:
```bash
# ccodex aliases - generated by npx ccodex
ccodex() {
  command npx -y ccodex "$@"
}

if ! typeset -f co >/dev/null 2>&1; then
  co() {
    command npx -y ccodex "$@"
  }
fi

claude-openai() {
  command npx -y ccodex "$@"
}
```

#### 3.2 Shell Integration

**Expected**: Source line present
